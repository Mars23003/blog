基本介紹

HTML是什麼?其中文全名為「超文字標示語言」

透過標籤與內容來讓瀏覽器知道要怎麼呈現你的網頁。

下面是基本範例:

<!DOCTYPE html>
<html>
<head>
<title>頁面標題</title>
</head>

<body>

<h1>這是一個h1的標題</h1>
<p>這是一個段落標籤</p>

</body>
</html>

範例連結

  1. <!DOCTYPE html> 告訴瀏覽器這是一個HTML5的網頁。
  2. <html> 也被稱為根元素(root element)，表示其包覆的內容為HTML文件，裡面會有<head></head>、<body></body>標籤。
  3. <head></head>裡面的內容不是給網頁瀏覽者看的，通常其對象是瀏覽器。
  4. <body></body>裡面的內容是給網頁瀏覽者看的。
  5. <h1></h1>h1大小的標題
  6. <p></p>一個段落標籤
