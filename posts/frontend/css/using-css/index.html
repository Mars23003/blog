[CSS]簡單範例了解CSS引用的優先順序

當瀏覽器讀取到CSS的樣式表時會透過樣式表內容來調整HTML的呈現，透過簡單範例了解CSS引用的優先順序。
September 7, 2022 · 1 分鐘 · 316 字 · Mars
目錄
  • CSS的引用有分為三種。

CSS的引用有分為三種。#

  1. 行內引用(Inline CSS)

在標籤內引用

<!DOCTYPE html>
<html>
<body>
<h1 style="color:blue">我是h1</h1>
</body>
</html>
  1. 內部引用(Internal CSS)

在<head></head>內的<style></style>引用

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
  color: blue;
}
</style>
</head>
<body>

<h1>我是h1</h1>

</body>
</html>
  1. 外部引用(External CSS)

指定外部檔案的路徑來應用

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="xxx.css"> <!-- xxx.css 寫檔案路徑 -->
</head>
<body>

<h1>我是h1</h1>

</body>
</html>

透過範例中的例子可以很清楚知道引用CSS的優先順序 順序為行內>內部>外部。
