[JavaScript] 立即執行函式(IIFE)是什麼?

IIFE (Immediately Invoked Function Expression) 是一個定義完就立即執行的 JavaScript function。 又稱為 Self-Executing Anonymous Function
September 14, 2022 · 1 分鐘 · 98 字 · Mars
目錄
  • 介紹

介紹#

主要有兩個部分:

  1. 透過()包住匿名函式來避免汙染到全域環境。
  2. 最後的()來立即執行function。
(function () {
    console.log("大家好")
})();

好處:

  1. 立即執行程式碼
  2. 避免汙染全域環境
