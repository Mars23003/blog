[Hugo] 使用Hugo創建網站超簡單!跟著做立刻上手!

Hugo 是一個快速、靜態網站生成器，它使用 Go 語言開發，支持 Markdown 語法、模板自定義和快速編譯等功能。Hugo 的主要特點是速度快、佔用資源少，可以在幾秒鐘內生成大型的靜態網站，非常適合用於個人部落格。

以下是使用 Hugo 的基本步驟：

安裝 Hugo#

首先需要在電腦上安裝 Hugo，可以從官方網站上下載對應平台的安裝包，或者使用指令進行安裝：

Mac#

  1. 開啟終端機Terminal
  2. 先安裝brew的套件管理工具(官方連結)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  1. 再透過以下指令進行Hugo安裝
brew install hugo

Windows#

  1. 設定PowerShell 先透過系統管理員權限運行PowerShell
# 輸入以下指令來做權限設定
Set-ExecutionPolicy Bypass -Scope Process

2. 確認PowerShell設定
# 透過以下指定取得當前狀態 若設定成功將取得bypass字串
Get-ExecutionPolicy
  1. 透過PowerShell安裝Chocolatey的套件管理工具(官方連結)
# 安裝Chocolatey
Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
  1. 透過PowerShell安裝Hugo
choco install hugo -confirm
# or
choco install hugo-extended -confirm

創建新網站專案#

使用 Hugo 創建新網站專案很簡單，只需要執行以下命令：

hugo new site mysite

這將在當前目錄下創建一個名為 mysite 的新專案。

增加主題#

Hugo 支持多種主題，可以在官方網站上下載主題，也可以使用CMD指令安裝：

cd mysite
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
echo 'theme = "ananke"' >> config.toml

增加內容#

在 mysite/content 目錄下增加 Markdown 文件，例如：

hugo new posts/my-first-post.md

編譯專案#

使用以下命令編譯專案：

hugo -D

這將在 mysite/public 目錄下生成靜態網站。

預覽網站#

使用以下命令啟動本地 Web server，預覽網站：

cd public
hugo server -D

部屬網站#

將 public 目錄下的內容上傳到 Web 服務器即可完成網站的部署。

以上是使用 Hugo 創建靜態網站的基本步驟，更多功能和使用細節可以參考 Hugo 的官方文件。
