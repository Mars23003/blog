Mermaid 是什麼？#

Mermaid 是一個用於生成圖表和流程圖的工具，支援 Markdown 語法，適合用於技術文件、部落格文章或專案文件中。

在 VS Code 中使用 Mermaid#

1. 安裝 Mermaid 擴充套件#

在 VS Code 中，搜尋並安裝名為 “Markdown Preview Mermaid Support” 的擴充套件。這個擴充套件允許你在 Markdown 文件中直接預覽 Mermaid 圖表。

2. 編寫 Mermaid 語法#

在 Markdown 文件中，使用以下語法來插入 Mermaid 圖表：

```mermaid
flowchart TD
    A[開始] --> B[進行中]
    B --> C[完成]


### 3. 預覽 Mermaid 圖表

按下 `Ctrl+Shift+V` 或右鍵選擇 **"Markdown: Open Preview"**，即可在 VS Code 中預覽 Mermaid 圖表。

## Mermaid 支援的圖表類型

Mermaid 支援多種圖表類型，包括但不限於：

1. **流程圖 (Flowchart)**
2. **序列圖 (Sequence Diagram)**
3. **甘特圖 (Gantt Chart)**
4. **類別圖 (Class Diagram)**
5. **狀態圖 (State Diagram)**

以下是一個序列圖的範例：

```mermaid
sequenceDiagram
    participant A as 使用者
    participant B as 系統
    A->>B: 發送請求
    B-->>A: 回應結果

Mermaid 的優勢#

  • 簡單易用：使用類似 Markdown 的語法。
  • 高度可定制：可以調整樣式和佈局。
  • 跨平台：支援多種工具和編輯器。

結論#

Mermaid 是一個強大的工具，特別適合用於技術文件和部落格文章中。透過 VS Code 的擴充套件，你可以輕鬆地在 Markdown 文件中創建和預覽圖表。
