入門操作：讓 Copilot 寫你的第一個函式#

啟用 Copilot 基本功能#

  • 確認 Copilot 已安裝並授權（見前一篇安裝指南）。
  • 開啟任意支援的程式語言檔案（如 .js、.ts、.py、.cs 等）。
  • 輸入註解或函式名稱，Copilot 會自動顯示建議。

撰寫第一個示範函式#

  1. 在程式碼區塊輸入註解，例如： 
    # 計算兩數相加的函式
  2. 按下 Enter，Copilot 會自動產生對應的函式程式碼。
  3. 按 Tab 接受建議，或 Esc 拒絕。

單行補全與多行生成#

  • 單行補全：在輸入一行程式碼時，Copilot 會即時顯示灰色建議，按 Tab 接受。
  • 多行生成：輸入函式註解或描述，按 Enter 讓 Copilot 產生多行完整程式碼。

快捷鍵與操作習慣#

  • Tab：接受 Copilot 建議
  • Esc：忽略建議
  • Alt+[ / Alt+]：切換不同建議（部分作業系統可能為 Ctrl 或 Option）
  • Ctrl+Enter：開啟 Copilot 面板，查看更多建議
  • 可於 VSCode 設定自訂 Copilot 快捷鍵

提示詞範例#

  • # 判斷質數的函式
  • // 反轉字串的 function
  • # 取得今天日期並格式化
  • // 產生隨機密碼
  • # 讀取 CSV 檔案並統計資料

多練習不同提示詞，能讓 Copilot 產生更貼近需求的程式碼片段！
