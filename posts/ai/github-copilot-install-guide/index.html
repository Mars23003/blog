【新手指南】如何在 Visual Studio Code 安裝 GitHub Copilot#

什麼是 GitHub Copilot#

GitHub Copilot 是由 GitHub 與 OpenAI 合作推出的 AI 程式碼產生器，能即時協助你自動補全、產生程式碼片段，提升開發效率。

安裝需求與準備#

  • 已安裝 Visual Studio Code（建議最新版）
  • 有 GitHub 帳號（建議已啟用雙重驗證）
  • 穩定的網路連線

安裝步驟逐步圖解#

  1. 開啟 VSCode，點選左側「擴充套件」圖示（或快捷鍵 Ctrl+Shift+X）。
  2. 搜尋「GitHub Copilot」，點選進入後按「安裝」。
  3. 安裝完成後，點選右下角彈出的 Copilot 啟用提示，依指示登入 GitHub 帳號並授權。
  4. 授權成功後，Copilot 會自動啟用，並可在程式碼編輯時看到 AI 補全建議。

啟用與授權#

  • 第一次使用需登入 GitHub 並授權 Copilot。
  • 若有 Copilot 訂閱，請確認帳號已啟用。
  • 可於 VSCode 右下角 Copilot 圖示檢查授權狀態。

常見問題排查#

  • 無法安裝/找不到 Copilot？
    • 請確認 VSCode 已更新至最新版。
    • 檢查網路連線與代理設定。
  • 授權失敗？
    • 確認 GitHub 帳號已啟用 Copilot 訂閱。
    • 嘗試重新登入或重啟 VSCode。
  • AI 補全沒反應？
    • 檢查 Copilot 是否啟用，可於命令面板輸入 Copilot: Enable。
    • 檢查檔案類型是否支援。

個人化設定技巧#

  • 可於 VSCode 設定中搜尋「Copilot」調整偏好（如自動建議、語言支援等）。
  • 支援自訂快捷鍵、啟用/停用特定檔案類型。
  • 善用 Copilot Labs 擴充功能，體驗更多 AI 實驗功能。
